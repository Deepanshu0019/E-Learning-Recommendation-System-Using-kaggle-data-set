# E-Learning-Recommendation-System-Using-kaggle-data-set
An interactive e-learning recommendation system built with Streamlit using Kaggle datasets. It applies machine learning to analyze learner behavior and course features, delivering personalized, real-time course suggestions through a clean and user-friendly web interface.
